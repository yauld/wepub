## 截图要求

- 使用浅色模式，宽度建议 1600px 以上。
- 遮住 AppID、AppSecret、access_token、真实公网 IP、账号主体信息等敏感内容。
- 如果公众号名称不想公开，也请遮住。
- 尽量使用真实技术文章，但避免露出未发布的商业信息。
- 截图文件建议控制在 1 MB 内。

## README 使用策略

- README 顶部放主工作台截图，快速说明产品形态。
- “产品截图”章节展示主工作台、同步弹窗和草稿箱结果，形成完整证据链。
- 公众号编辑器截图最能建立信任，但如果包含账号名，发布前应先确认或脱敏。
